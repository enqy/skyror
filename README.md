# skyror ISA

A minimal RISC inspired ISA.

## skyror32

The 32-bit version of skyror.

[Spec](./spec32.adoc)

## Building the PDF Version

1. Install [Nix](https://nixos.org/download.html)
2. `nix build .#skyror32` for the 32-bit version
